Simple Steps to Set Up and Use Your Extension

Step 1: Download and Unzip the Extension

  • Download the extension file (in .zip format) from the provided link
  • Right-click the downloaded file and select "Extract All" (or similar option on your computer)
  • Note the folder location where the unzipped files are saved

Step 2: Go to Chrome Extensions Page

  • Open Chrome and type chrome://extensions in the address bar
  • Press Enter to navigate to the Extensions page

Step 3: Enable Developer Mode

  • On the Extensions page, look at the top right corner
  • Toggle "Developer mode" on (this allows you to load extensions from folders)

Step 4: Upload the Extension

  • Click on "Load unpacked" at the top of the Extensions page
  • Navigate to the folder where you unzipped the extension in Step 1
  • Select the folder and click "Open"
  • The extension will load, and you'll see its icon appear in Chrome

Step 5: Pin the Extension

  • Click on the puzzle icon in the top-right corner of Chrome
  • Find your extension in the list and click the pin icon next to it
  • This makes it visible on the Chrome toolbar

Step 6: Open the Extension and Log In

  • Click on the pinned extension icon to open it
  • Follow the on-screen prompts to Log In or Sign Up
  • You're all set! The extension is now ready to use
